Monitors your internet connection in real time

Usage: 
```
chmod +x watchnet.sh

./watchnet.sh
```
Type one of the listed network interfaces, type CTRL + C when you want to quit.

For Arch Linux users: 
```
git clone https://github.com/cryptrz/freecache.git

cd freecache

makepkg -si

sudo pacman -U freecache-1.0.0-1-any.pkg.tar.zst

sudo freecache
```
